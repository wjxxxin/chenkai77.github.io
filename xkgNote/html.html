基本标签

代码

h1~h4(标题)

  1. 用来显示标题
  2. h1最大，h4最小
  3. 一个网页中最好只有一个h1，作为最主要的标题

p（段落）

  1. 用来显示一个段落

pre（预格式化的文本）

  1. 被包围在 pre 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。标签的一个常见应用就是用来表示计算机的源代码。

img（图片）

  1. 用来显示一个图片
  2. src :设置图片的URL
  3. width： 图片的宽度
  4. height：图片的高度
  5. title：设置图片在鼠标移上去时显示d额描述文本
  6. alt：占位文字（当图片显示失败的时候显示这个占位文字）

a（链接）

  1. 表示超链接标签
  2. href：表示要链接的地址(URL)。（前面添加http://协议符号） 可用于设置锚点链接，如下 
    <a href="#id"></a>
  3. target:在哪里打开
    1. _self:在当前页面打开（覆盖当前页面）
    2. _blank:在新页面打开
    3. _parent：在父页面打开
    4. _top：在顶层页面打开
    5. 某个iframe的name属性值：在某个iframe中打开

列表标签

  1. ul>li*X:无序列表标签
  2. ol>li*X:有序列表标签
  3. dl>dt+dd*X:项目列表标签

语义化标签

  1. header 头部
  2. section 区块（在h5中一般高于div）
  3. nav 导航
  4. figure 媒体（图像，图表，代码等等）
  5. figcaption 媒体标题
  6. aside 侧边栏
  7. article 文章（一般适用于文字列表或者正文）
  8. footer 底部

表单标签

input 它可以根据自己type属性值的不同来变换外观和功能。

  1. type 类型
    1. text：文本
    2. password：密码
    3. button：普通按钮 
      <button value="哇哈哈">我是新版按钮</button>
    4. radio：单选框
    5. checkbox：复选框
    6. submit：发送
    7. reset：重写
    8. file：上传
    9. image：图片
    10. email：邮箱
    11. url：网址
    12. search：搜索框
    13. color：拾色器
    14. range：范围
  2. value：框上预设文字
  3. placeholder：占位文字
  4. name：为了防止单选的时候，用户多项选择，只需给不同的input标签设置相同的name属性值即可，复选框 name属性不会限制用户多选。

label标签

扩大表单元素的触发范围。标签属性for，id；给这个标签添加一个for属性，属性值是某个表单元素的id即可。新版：直接用label把文字信息和表单元素包含起来就可以了。

fieldset legend 标签

文本域

textarea

下拉菜单元素

  1. select>option: 默认是显示option第一项，如果要指定默认项，只需要给option添加selected属性即可。
  2. 设置multiple="multiple" ; 可设置多选。

音频 视频标签

video 视频标签

  1. src：视频路径（必选属性）
  2. - controls：控制条，表示是否添加播放控件；（一般情况这个都是不开放给用户的）
  3. autoplay：是否自动播放，在谷歌浏览器中是不能自动播放的，在最新的火狐浏览器也是默认被禁用，需要手动打开
  4. width：宽度
  5. height：高度
  6. loop：设置视频是否循环播放；（广告肯定有这个属性）
  7. preload：预加载（当具有preload的时候，autoplay会相冲突，不起作用）
  8. poster：视频封面
  9. muted：静音
<video src="media/super.mp4" controls width="500" poster="images/yx.jpg" autoplay loop preload muted>
</video>

source：设置video标签的多个格式的视频。浏览器会优先播放支持的格式。使用的基本结构：video>source 相当于又是一个 ul>li结构；标签属性：src：视频路径；type：表示设置视频的格式

audio 视频标签

audio几乎拥有所有的video属性，除了3个可见属性：width，height，poster。

命名收集

  1. wrap：vt. 包；缠绕；隐藏；掩护 ；vi. 包起来；缠绕；穿外衣；n. 外套；围巾
  2. item：n. 条款，项目；一则；一件商品（或物品）；adj. 又，同上；v. 记下；逐条列出
  3. slogan：n. 标语；呐喊声
  4. wrapper：包裹，（版芯）
  5. main：主要的（版芯）
  6. con：（container）内容、集装箱（版芯）
  7. topbar：顶部条，顶部标题
  8. category：种类，分类
  9. caret：插入符号

常用转义字符

  1. 空格：&nbsp;
  2. 版权号: &copy;
  3. 大于号：&gt;
