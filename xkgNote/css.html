css书写格式

css的书写格式有三种：内嵌式，内联式，外联式

CSS样式优先级计算法:
  1. 就近原则,后加样式优于前面的样式
  2. 内嵌样式>内联样式>外联样式（内联式和外联式哪个距离html对象越近，优先级越高。）
  3. !Important 大于一切样式

css两个最大特性

  1. 层叠性：后面的属性会覆盖前面写的属性；
  2. 继承性：给父级写的样式子级标签也能生效，文字控制属性是可以继承的，超链接的文本文字属性除外

css单位

  • em ： em 单位是相对于使用em单位的元素的字体大小( 也是相对于字体大小，有可能是父亲的元素也有可能是自身的元素字体大小。（如果一个 div 有 18px 字体大小，给标签10em 将等同于 180px，即10em × 18px = 180px）
  • rem ： Rem转化为像素px大小取决于页根元素html的字体大小，即 html 元素的字体大小
  • vw ：相对于视窗的宽度：视窗宽度是100vw
  • vh ： 相对于视窗的高度：视窗高度是100vh

定位

  4. z-index：（定位元素的图层顺序调节属性） 控制定位元素的上下显示关系，注意它只是表示一种图层顺序所以不带px单位。

相对定位是相对于网页进行定位；绝对定位是相对于最近定位的元素进行定位，否则相对窗口进行定位

边距属性

  1. margin：外边距；表示标签以外的距离
  2. padding：内边距；表示标签与内容之间的距离

    单独指定某个方向的边距：
  1. margin-top
  2. margin-right
  3. margin-bottom
  4. margin-left

    用1~4个值书写内外边距时表达的意思：
  1. 1个值：表示四个方向的距离
  2. 2个值：表示上下、左右的距离
  3. 3个值：表示上、左右、下的距离
  4. 4个值：表示上、右、左、下的距离（顺时针）

    box-sizing：
  1. border-box：为元素指定的任何内边距和边框都将在已设定的宽度和高度内进行绘制。通过把已设定的 'width' 和 'height' 属性分别减去边框和内边距才能得到内容的宽度和高度。
  2. content-box：默认值

    外边距塌陷现象：
  1. 两个同级的盒子紧挨着出现垂直边距，两个外边距margin会相互粘连，以最大的边距进项计算；解决方法：尽量避免同时出现两个都是外边距的情况，可以只给一个盒子增加margin；
  2. 两个嵌套的盒子，父级盒子和子级盒子同时都存在同一方向的边距，两个边距就会相互粘连，以最大的边距进项计算；解决方式：给父级添加一个css样式overflow：hidden；

清除默认边距：有些标签默认带有内外边距，如标题标签、段落标签，可以使用通配符——*选中所有标签；清空默认边距时需要清空 padding，margin，border将它们的值都设为0即可。

背景属性：background

  1. background-color ： 背景色
  2. background-image ： 背景图片
  3. background-repeat ： 平铺方式 ：no-repeat 不平铺；repeat-x 横向平铺；repeat-y 纵向平铺
  4. background-position ： 水平位置 垂直位置 ： 水平位置的值有：left center right；垂直位置的值有：top center bottom ；还有一种是给具体的坐标值：例如：100px 200px （坐标值接受负值；应用背景图片标签的左上角是坐标为0 0的原点）
  5. 固定方式 固定方式如果设置为fixed，则背景图片固定在网页某个位置 
    background：#ccc url(dog.jpg) no-repeat left top fixed
  6. 设置多个背景可用逗号隔开，写在前面的背景会显示在前面 
    background: url("../images/con_bg_l.jpg") no-repeat,
            url("../images/con_bg_center.jpg") repeat-x,
            url("../images/con_bg_r.jpg") no-repeat;
    设置背景时先指定背景图再指定背景颜色才能把图片显示出来（显示在颜色上）
  7. inherit : 背景跟随父元素
  8. background-size：
    1. 可以接受百分比设置。
    2. 可以接受固定的人为像素设置尺寸。
    3. 可以接受两个关键词设置。
      1. contain：表示尽可能完整的显示图片。
      2. cover：表示尽可能铺满盒子。
  9. background-origin：背景起源属性
    1. border-box：从边框盒子开始渲染图片。
    2. padding-box：从内边距盒子开始渲染图片。
    3. content-box：从内容空间盒子开始渲染图片。
  10. background-clip ： 背景裁切属性
    1. border-box
    2. padding-box
    3. content-box
    默认情况背景起源和背景裁切它的值都是padding-box
  11. background：linear-gradient(180deg,red,green,blue);渐变颜色;其中如果色值的后面跟一个百分比表示染色的位置。如果不写表示均分颜色;0deg表示从下到上，180deg表示从上到下

溢出属性

    overflow:
  1. visible 默认值。内容不会被修剪，会呈现在元素框之外。
  2. hidden 内容会被修剪，并且其余内容是不可见的！还可用来处理外边距塌陷（嵌套关系）问题，会触发BFC
  3. scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。（横向竖向的都会有）
  4. auto 如果内容被修剪，那么浏览器会显示滚动条以便查看其余的内容。（只会根据情况出现滚动条）
  5. inherit 规定会从父元素继承overflow属性的值

横向滚动:父元素:overflow-x:auto; overflow-y:hidden;

竖向滚动:父元素 overflow-x:hidden;overflow-x:auto;

选择器

基本选择器

  1. 标签选择器：使用标签进行选择
  2. 类选择器：使用类进行选择 class=""
  3. id选择器：使用id进行选择
  4. 后代选择器：用一个空格表示“里面的” 的意思，作用是通过父级选中它内部的后代标签； 例如：ul li 表示选中 ul里面的所有li； .box div 表示选中类名为box 里面的 div；
  5. 并集选择器：多个选择器之间用逗号隔开表示共同关系；作用是可以将多个选择器选中共同来给它们写样式；选择器不分先后顺序；例如:div,p,.box,#box{}
  6. 交集选择器：通过标签名称和标签身上的类名连续书写（中间不需要空格或符号，连续写；）作用是用来选中带有类名的那个标签；例如：div.box 表示只选中带有 box类名的那个div ；
    7. 选择器权重： id：1吨；类：1斤；标签：1克；通配符：1毫克；

高级选择器

  1. 子级选择器：> ：表示父元素的直接子级元素（亲儿子）
  2. 同级选择器：+ ：表示同级元素紧挨着的下一个元素
  3. ~ ：选择前面有同级元素的所有兄弟级元素
  4. 属性选择器：
    1. [k]：标签中包含k属性的选择器（div[id] 表示标签中含有id属性的div）
    2. [k=“v”]：表示属性为k，属性值为v的选择器
    3. [k^=“v”]：表示属性为k，属性值为v开头的选择器
    4. [k$="v"]：表示属性为k，属性值为v结尾的选择器
    5. [k*="v"]：表示属性为k，属性值中包含v的选择器
    特殊选择器
  1. ：root ：表示选择HTML根标签
  2. ：empty ：表示选择空标签（标签内容为空）
  3. ：not（k）：表示非k这个标签
  4. ：selection ：鼠标框选中的元素（火狐没反应）
    个数选择器
  1. 同级选择器（重个数）
    1. n:nth-child（k） ：同级中的第k个n元素
    2. n:first-child ：同级中第一个n元素
    3. n:last-child ：同级中最后一个n元素
    4. n:nth-last-child（k） ：同级中倒数第k个n元素
    同级选择器如果后面个数为k的标签与前面的n不符合则找不到
  2. 同类型选择器（重类型）
    1. n:nth-of-type（k） ：同类型中第k个元素
    2. n:first-of-type ： 同类型中第一个元素
    3. n:last-of-type ： 同类型中最后一个元素
    4. n:nth-last-of-type(k) : 同类型中倒数第k个元素

投影 圆角 不透明度

box-shadow : 盒子投影属性

box-shadow:20px （水平位置） 30px（垂直位置） 40px（模糊程度） 50px（阴影大小） #000（阴影颜色） inset（内外位置，默认是外面）

box-shadow：至少有两个值，第一个表示水平位置，第二个表示垂直位置，投影默认并不是黑色，实际上是跟随了文本的颜色，模糊程度不能为负数。

text-shadow：文字投影属性

text-shadow：10px（水平位置） 20px（垂直位置） 30px（模糊程度） green投影颜色

text-shadow：至少要有2个值，第一个表示水平位置，第二个表示垂直位置，默认投影是黑色模糊程度不能为负数。

border-radius：圆角属性

1个值：四个角
2个值：左上右下 右上左下
3个值：左上 右上左下 右下
4个值：左上 右上 右下 左下
半圆：宽度和高度关系2:1，圆角第一个值和第二个值为宽度的一半（高度的值）,其他给0
正圆：宽度和高度1:1，圆角只要一个值，为宽度的一半，或者50%

不透明度

  1. opacity：元素不透明度的效果 （值可以是0-1的小数，小数点之前的0可以省略，也可以是0-100%的百分数，0表示完全透明，但是不是消失，还会占用显示的时候的位置）
  2. rgba：只是颜色的属性值,表示颜色的色值的方式有：英文单词，16进制，rgb（rgba），hsl（hsla），rgb的值范围是0-255， 0,0,0 黑色 255,255,255白色 
background: rgb(128,0,0);
/*a表示alpha 透明度的意思，数值0-1的小数，0可以省略*/
background: rgba(255,0,0,.5);

opacity能够让背景和内容同时透明，rgba只能给元素自身（color）或者背景（background）透明

transform： 转换属性

  1. rotate（xdeg） ：旋转

    transform-origin：用来设置盒子的旋转中心点

    它可以接受三种类型的值（像素类型的值；百分比；特殊关键词），默认是先水平 垂直。

  2. scale（1,1） ：它的小括号里有两个值表示水平缩放和垂直缩放，其中1表示缩放为100%也就是表示不变。
  3. translate（水平偏移，垂直偏移）： 位移
    4. 以上三大功能都可以在最后加一个大写的X,Y,Z后缀来单独设置是哪个方向。
  4. skew（水平度数，垂直度数）: 倾斜功能；注意事项：度数越大会被拉得越长

transition：过渡属性

transition：属性名称 过度时间 动画类型缓动 延迟时间。

其中属性名称可以通过一个all来设置所有。动画类型要特殊记住linear表示匀速；其他的属性中可能transition-delay会单独用到。

自定义动画属性

animation：设置自定义的动画

    自定义动画步骤
  1. 选择一个元素通过animation绑定动画并设置时间
  2. 通过@keyframes name{}关键词来声明一个动画。动画名是自定义的不能以数字开头。
  3. 在大括号的内部通过百分比定义动画的关键帧，每个关键帧都是一个样式条。直接书写这个状态的css样式即可。
  4. 如果只有两个变化，0%可以写为from，100%可以写成to；

animation ：动画名称 动画时间 动画缓动 动画延迟 动画次数 动画方向 播放状态
  1. 动画缓存中linear表示匀速；
  2. 动画次数中infinite表示无限次；
  3. 动画方向中alternate表示往复播放；
  4. 播放状态中paused：暂停 running：播放；
  5. animation-play-state表示播放状态单独控制。

动画定义符

三维参照属性

transform-style：preserve-3d

三维的坐标系需要给父级设置一个三维参照属性，才能让子级顺利进入三维空间，这个原理类似于子绝父相。

在使用transform属性的时候要避免被后面的动画覆盖，如果涉及到用缩放功能尽量将缩放放在最后书写，最理想的顺序：transform：位移 旋转 缩放；

手写动画的最小时间间隔

多数显示器默认频率是60Hz，即1秒刷新60次，所以理论上最小间隔1/60*1000ms = 16.7ms

flex布局

display:flex/inline-flex

设为 Flex布局后，子元素的float、clear和vertical-align属性将失效。但是position属性，依然生效。

父元素属性

  1. flex-flow flex-direction和flex-wrap的缩写形式
    1. flex-direction 设置主轴方向
      1. row（默认值）： 主轴为水平方向，起点在左端
      2. row-reverse： 主轴在水平方向，起点在右端
      3. column：主轴为垂直方向，起点在上沿
      4. column-reverse：主轴为垂直方向，起点在下沿
    2. flex-wrap
      1. nowrap（默认）：不换行。当容器宽度不够时，每个项目会被挤压宽度
      2. wrap： 换行，并且第一行在容器最上方
      3. wrap-reverse： 换行，并且第一行在容器最下方
  2. align-content 配合flex-wrap使用
    1. stretch（默认值）：轴线占满整个交叉轴
    2. flex-start：与交叉轴的起点对齐
    3. flex-end：与交叉轴的终点对齐
    4. center：与交叉轴的中点对齐
    5. space-between：与交叉轴两端对齐，轴线之间的间隔平均分布
    6. space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍
  3. justify-content 项目在主轴上的对齐方式
    1. flex-start（默认值）： 项目位于主轴起点。
    2. flex-end：项目位于主轴终点。
    3. center： 居中
    4. space-between：两端对齐，项目之间的间隔都相等。(开头和最后的项目，与父容器边缘没有间隔)
    5. space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。(开头和最后的项目，与父容器边缘有一定的间隔)
  4. align-items 定义项目在交叉轴上如何对齐。
    1. flex-start：交叉轴的起点对齐。(默认)
    2. flex-end：交叉轴的终点对齐。
    3. center：交叉轴的中点对齐。
    4. baseline: 项目的第一行文字的基线对齐。(文字的行高、字体大小会影响每行的基线)
    5. stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

子元素属性

  1. order 定义项目的排列顺序。数值越小，排列越靠前，默认为0。
  2. flex flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。
    1. flex-grow 定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。设为非0的子元素按数值平均分配。
    2. flex-shrink 定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。设为0则不缩小。
    3. flex-basis定义项目占据的主轴空间。(如果主轴为水平，则设置这个属性，相当于设置项目的宽度。 原width将会失效。)
  3. align-self:定义单个项目自身在交叉轴上的排列方式，可以覆盖掉容器上的align-items属性。 属性值与align-items相同，默认值为auto，表示继承父容器的align-items属性值。

grid布局

display:grid/inline-grid

设为网格布局以后，容器子元素（项目）的float、display: inline-block、display: table-cell、vertical-align和column-*等设置都将失效。

网格容器(Grid Container) 属性

  1. grid-template-columns :<track-size> ... | <line-name> <track-size> ...
  2. grid-template-rows : <track-size> ... | <line-name> <track-size> ... 
    .container {
  grid-template-columns: [first] 40px [line2] 50px [line3] auto [col4-start] 50px [five] 40px [end];
  grid-template-rows: [row1-start] 25% [row1-end] 100px [third-line] auto [last-line];
}
    1. track-size：定义网格单元的宽高，其单位可以是一个长度(如px、em、rem、vw、vh)或百分比，也可以是网格中自由空间的份数(单位为fr)。fr 单元是除去所有非灵活网格项之后的剩余空间等分。
    2. line-name：定义网格线的名称，它不是必须值。可以一个你选择的任意名字，当没有显示设定时，它的名字以数字表示。一条网格线(Grid Line)可以有多个名称,例如 grid-template-rows: [row1-start] 25% [row1-end row2-start] 25% [row2-end];
      3. repeat() 可以简化重复的值,接受两个参数，第一个参数是重复的次数（上例是3），第二个参数是所要重复的值。 例如 : grid-template-columns: repeat(2, 100px 20px 80px);
  3. grid-template-areas
    1. grid-area-name：由网格项的 grid-area 指定的网格区域名称
    2. .（点号） ：代表一个空的网格单元
    3. none：不定义网格区域
    .item-a {
  grid-area: header;
}
.item-b {
  grid-area: main;
}
.item-c {
  grid-area: sidebar;
}
.item-d {
  grid-area: footer;
}

.container {
  grid-template-columns: 50px 50px 50px 50px;
  grid-template-rows: auto;
  grid-template-areas:
    "header header header header"
    "main main . sidebar"
    "footer footer footer footer";
}

    声明中的每一行都需要有相同数量的单元格。

    区域的命名会影响到网格线。每个区域的起始网格线，会自动命名为区域名-start，终止网格线自动命名为区域名-end。

    比如，区域名为header，则起始位置的水平网格线和垂直网格线叫做header-start，终止位置的水平网格线和垂直网格线叫做header-end。

  4. grid-gap grid-column-gap和grid-row-gap的合并简写形式
    1. grid-row-gap用于设置行间距
    2. grid-column-gap用于设置列间距。
  5. grid-auto-flow
    1. row : 默认值 "先行后列"
    2. column : "先列后行"
    3. row dense
  6. place-items align-items属性和justify-items属性的合并简写形式
    1. justify-items 设置单元格内容的水平位置（左中右）/ align-items 设置单元格内容的垂直位置（上中下）
      1. start：对齐单元格的起始边缘。
      2. end：对齐单元格的结束边缘。
      3. center：单元格内部居中。
      4. stretch：拉伸，占满单元格的整个宽度（默认值）。
  7. place-content align-content属性和justify-content属性的合并简写形式。

    如果用像px非弹性单位定义的话，总网格区域大小有可能小于网格容器，这时候你可以设置网格的对齐方式

    1. justify-content 整个内容区域在容器里面的水平位置（左中右） / align-content 整个内容区域的垂直位置（上中下）
      1. start : 对齐容器的起始边框。
      2. end : 对齐容器的结束边框。
      3. center : 容器内部居中。
      4. stretch : 项目大小没有指定时，拉伸占据整个网格容器。
      5. space-around ：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与容器边框的间隔大一倍。
      6. space-between : 项目与项目的间隔相等，项目与容器边框之间没有间隔。
      7. space-evenly : 项目与项目的间隔相等，项目与容器边框之间也是同样长度的间隔。
  8. grid-auto-columns / grid-auto-rows 浏览器自动创建的多余网格的列宽和行高。写法与grid-template-columns和grid-template-rows完全相同。如果不指定这两个属性，浏览器完全根据单元格内容的大小，决定新增网格的列宽和行高。

网格项(Grid Item)属性

  1. grid-column 属性， grid-row 属性

    grid-column属性是grid-column-start和grid-column-end的合并简写形式，grid-row属性是grid-row-start属性和grid-row-end的合并简写形式。

    这两个属性之中，也可以使用span关键字，表示跨越多少个网格。

    .item-1 {
  background: #b03532;
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}

    斜杠以及后面的部分可以省略，默认跨越一个网格。

    .item-1 {
  grid-column: 1;
  grid-row: 1;
}

    上面代码中，项目item-1占据左上角第一个网格。

    如果产生了项目的重叠，则使用z-index属性指定项目的重叠顺序

  2. grid - area :指定项目放在哪一个区域。

    grid-area属性还可用作grid-row-start、grid-column-start、grid-row-end、grid-column-end的合并简写形式，直接指定项目的位置。

    .item-1 {
  grid-area: 1 / 1 / 3 / 3;
}
  3. place-self align-self属性和justify-self属性的合并简写形式。

    justify-self属性设置单元格内容的水平位置（左中右），跟justify-items属性的用法完全一致，但只作用于单个项目。

    align-self属性设置单元格内容的垂直位置（上中下），跟align-items属性的用法完全一致，也是只作用于单个项目。

    1. start：对齐单元格的起始边缘。
    2. end：对齐单元格的结束边缘。
    3. center：单元格内部居中。
    4. stretch：拉伸，占满单元格的整个宽度（默认值）。

浮动塌陷、外边距塌陷

浮动塌陷

当父级没有高度，内部元素浮动的时候，父级就会因为获取不到元素内部的高度显示异常，那么要想恢复的方式是，给父级添加overflow：hidden；属性，作用就是检测内部浮动元素。

浮动元素会导致父级没有高度或者高度为0.导致原因：父级没有指定高度，子元素又浮动

    解决方法：
  1. 给父级添加overflow：hidden;
  2. 给父级内部(子元素并级)添加一个元素，高度为0，且溢出隐藏，字体大小为0，并且要清除浮动。
  3. 使用伪元素清除浮动

外边距塌陷

    1、嵌套结构之间，父元素有上外边距，子元素有上外边距，会产生重叠，以最大的值进行显示。并且，嵌套之间并没有产生分离。
  1. 第一种解决方法：给父元素添加overflow:hidden；让父元素重新去识别内容。
  2. 第二种解决方法：给父元素添加clearfix清除浮动这种办法不过要注意,清除浮动重在清除前面的浮动

less

  1. less变量 : 通过@变量名称

  3. less运算 : less可以进行属性值的运算，例如宽高 文字大小 颜色等等数值和颜色之间的运算。(运算符与值之间必须以空格分开 ; 在运算的时候，只要有一个有单位就行)

calc() 函数

calc()支持CSS3；任何长度值都可以使用calc()函数进行计算；calc()函数支持 "+", "-", "*", "/" 运算; calc()函数使用标准的数学运算优先级规则

  1. 需要注意的是，运算符前后都需要保留一个空格，例如：width: calc(100% - 10px)；
  2. 任何长度值都可以使用calc()函数进行计算；
  3. calc()函数支持 "+", "-", "*", "/" 运算；
  4. calc()函数使用标准的数学运算优先级规则；
